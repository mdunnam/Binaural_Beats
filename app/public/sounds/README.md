# Soundscape Audio Files

Drop looping audio files here to enable real soundscape audio.
The app will automatically use these files instead of generated noise.

## Required files (OGG preferred, MP3 fallback):
- rain.ogg / rain.mp3
- thunder.ogg / thunder.mp3
- wind.ogg / wind.mp3
- waves.ogg / waves.mp3
- fire.ogg / fire.mp3
- forest.ogg / forest.mp3
- space.ogg / space.mp3
- cave.ogg / cave.mp3
